Portfolio

Project

Ketahui lebih lanjut beberapa proyek yang telah berhasil dibangun oleh Willian dengan hasil pembelajaran yang dilakukan pada masa kuliah dan pelatihan maupun kursus.

Melanoma Detection Using Backpropagation

Merupakan proyek Tugas Akhir dalam menggapai gelar Sarjana, sebuah sistem untuk mendeteksi penyakit kulit Melanoma dengan Jaringan Saraf Tiruan.

SPPD DISKOMINFO Kota Binjai

Sistem pembuatan Surat Perintah Perjalanan Dinas Secara otomatis sebagai bentuk proyek akhir pada Kerja Praktek di DISKOMINFO Kota Binjai. Dibangun dengan menggunakan PHP, HTML, CodeIgniter3.

QUOTES App

Proyek akhir pada praktikum pemrograman Mobile yang menunjukkan Quotes dari tokoh terkenal menggunakan bahasa Java dan menerapkan API serta library.

Sentiment Analyst Vaccine Covid-19

Mengukur respon masyarakat Indonesia akan kehadiran Vaksi Covid-19 dengan kategori Positif, Netral, dan Negatif. Dengan pengelolaan teknik scrapping melalui kutipan pengguna twitter.

Lens Birthday

Sebuah lens-snapchat yang dibangun dengan Lens Studio hasil pembelajaran kursus online Dicoding Indonesia. Menerapkan fitur segmentasi, visual scripting dan face effects.
